# Clock and Stopwatch Project

This is a small web project that shows a digital clock and a stopwatch. I made this to practice JavaScript basics and understand how time related functions work in real time.
![Uploading Screenshot 2026-04-15 at 10.27.16 PM.jpg…]()

## Features

- Live digital clock updating every second
- Stopwatch with start stop and reset
- Simple and clean UI
- Easy switch between clock and stopwatch
![Uploading Screenshot 2026-04-15 at 10.27.36 PM.jpg…]()

## Tech Used

- HTML
- CSS
- JavaScript

## How it works

The clock uses the JavaScript Date object to get current time and updates it every second.

The stopwatch works using setInterval where a counter increases every second and gets converted into hours minutes and seconds.

## What I learned

- How setInterval and clearInterval work
- Updating elements using DOM
- Basic time calculations
- Structuring a small frontend project
